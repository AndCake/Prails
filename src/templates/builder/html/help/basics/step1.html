Website basics

In order to create a static web page, we first need to create a container where it belongs to. Such a container is called a Module.

On the left-hand side of your screen, you can see a list of Modules. Currently there is only one, the Global module. The global module can't be changed. For our page we'll need to create a new module.

To do this, first click the Add button. In the center of the window, it will ask you for the module's name. Enter Bakery and click at the Save button.
