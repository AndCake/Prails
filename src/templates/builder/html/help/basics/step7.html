Website basics

As you might remember, the first part of the style definition is always the HTML tag to be styled. Until now we just used the body tag.

Now we want to add another style definition to our list. Let's make the headline bigger. Therefore we add another style definition in line 1: h1 { font-size: 32px; }. The H1 here means, that we want to change the style of the H1 HTML tag. The attribute we want to change is font-size and the size of the text should be 32 pixels.
