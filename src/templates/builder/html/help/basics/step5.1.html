Website basics

This style definition is called CSS - short for Cascading Style Sheets. It is used to describe the design of the page.

From what we wrote into the style tag, the word "body" means that we want to style all the HTML tags named body on the page. body here refers to an HTML element that always contains all the page's contents. Everything in our output code is in that body tag implicitly. That's why we didn't have to write it down in our HTML. The text within the curly braces is usually a set of attributes. In this instance, we defined that the body's background color attribute should be set to the given color.
