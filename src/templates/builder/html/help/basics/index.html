Website basics

All websites are basically documents that are sent from the web server to the client, normally the user's browser (e.g. Safari, Chrome or Firefox). This is called the "request-response model". The browser requests documents via a specific internet address (URL) and the server responds by sending it to the client.

This Prails installation is currently running on such a web server. There are several different kinds of web pages: static ones, that usually stay the same and are just written once for everyone to see. And then there's dynamic web pages, which contain for example user-related information or interactivity.

We will first discuss how to create a static page.
