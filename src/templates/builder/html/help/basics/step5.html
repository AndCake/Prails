Website basics

That was simple, wasn't it? Using HTML we declared the page structure. We will learn more about HTML later on. Now we will take a brief look at the page's design.

Therefore we add a new HTML tag just before the H1 begin tag type: <style>. As you might have noticed while typing, the editor closed that tag automatically for you. The style tag is there to contain style information for the HTML tags.

Between the opening and the closing style tags, write the following text: body { background-color: purple; }. Test what the result looks like by hitting the Save button and changing back to the "Bakery Home" tab.

Experiment a little with different color names. Try red, lightblue, teal or any other color name that comes to your mind.
