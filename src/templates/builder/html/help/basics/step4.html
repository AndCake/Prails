Website basics

Let's take a deeper look into what our generated event handler does. Therefore, double-click the home event handler in the left navigation menu. The event handler's details will appear in the center.

At first this might seem a bit confusing. For now, we will just focus on the second column's content that you can find below Output Code. So click at the + button right of the "Output Code" headline.

As you might notice, it contains the text that was shown when we ran the event handler. But in addition to that, there are some colored, slightly cryptic texts as well: the <h1> and </h1>, for example. This is called a Tag, which marks the beginning and end of a level 1 headline. Thereby the <h1> marks the beginning and the </h1> marks the end of it.

Similar, the <p> tag marks the beginning of a paragraph, whereas </p> marks the end of the paragraph. This kind of text is called "HTML", which means Hypertext Markup Language.
