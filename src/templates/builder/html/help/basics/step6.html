Other than color names, you can also use a very precise method: color codes. These usually consist of three values: one for defining how much red there should be in the color, one for the amount of green and one for blue. These values are mixed by the computer in order to get the actual color. This works just as it does in your paint palette.

In order to use color codes, there is a special notation: rgb(red, green, blue), whereas red, green and blue can be numbers between 0 (not in the resulting color) and 255 (maximum amount of that color is in the result).

First we've done the background-color attribute. The second attribute that we want to change now is called color. Therefore, add a second attribute just behind the semicolon: color: rgb(255, 255, 0);. Now again, test it and see how it looks like.
