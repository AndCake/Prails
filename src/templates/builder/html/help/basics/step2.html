Website basics

In Prails every request sent by the browser to get a page is handled by a so-called event handler. With the creation of the new module, Prails already generated our first event handler for us. Double-click the module name ("Bakery") on the left-hand side below the "Global" module and it will open up all the event handlers that do exist for the selected module.

As you then can see the generated event handler is called home. Let's see what it does. When you proceed with the following steps, a new tab, called "Bakery Home" will open in your browser, showing you what the event handler sends back to the browser. Select the home event handler and click the Run button above of it.
