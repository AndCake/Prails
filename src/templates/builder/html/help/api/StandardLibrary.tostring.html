tostring

Description:

convert an Array variable to a string (formatted for output)

Syntax:

String tostring($arr_var)

$arr_var

Array to be converted to a string

Returns: A formatted string representing the variable

Example:

            echo tostring(
            	Array(
            		Array(
            			"test" => "1234", 
            			"simple"=>"yes"
            		), 
            		Array(
            			"new" => Array(
            				"test", 
            				"simple"
            			), 
            			"now"=>time()
            		)
            	)
            );

This example will output the array defined as parameter.
