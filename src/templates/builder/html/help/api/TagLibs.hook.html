hook

Description:

This tag is used to call any hooks attached to the given hook name. Their result is directly printed out at the tag's position.

Syntax:

{hook name}

All hooks with that name will be notified and receive the current variable scope.

Example:

This example provides a place where other event can be hooked into: 

        	
My headline

Other events declared as hooks of the name "additional-text" can then add their own data at the specified position (by simply returning what should be added there).
