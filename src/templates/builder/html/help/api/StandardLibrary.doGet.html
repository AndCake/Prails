doGet

Description:

Does a server-side get request to another server/site. HTTPS is supported.

Syntax:

String doGet($str_url)

$str_url

URL to fetch data from

Returns: A string containing the data fetched from the (other) server

Example:

            echo doGet("http://www.google.com/#q=The+question+of+life+the+universe+and+everything");

This example will output Google's response on a search query.
