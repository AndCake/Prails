insert

Description:

inserts a tupel into the specified table and returns the ID of the new entry.

Syntax:

Integer insert($str_table, $arr_data)
Integer ins($str_table, $arr_data)
Integer add($str_table, $arr_data)
Integer set($str_table, $arr_data)

$str_table

The table's name the tupel should be inserted into.

$arr_data

Data to be inserted. Only entries, whose key matches one of the table's field names, are actually taken into account.

The ins(), add() and set() methods are aliases for the insert() method.

Example:

This example inserts a user into the user table. 

            $user_id = $this->add("user", Array(
                "first_name" => "Test",
                "last_name" => "User",
                "email" => "tester@example.org"
            ));
