wordCut

Description:

Function to cut text after a specified number of characters. With this function words remain complete (it won't be cut in the middle of a word).

Syntax:

String wordCut($str_text, $int_limit, $str_app)

$str_text

Text to cut

$int_limit

Maximum number of characters after which to cut

$str_app

Text to be appended after the text has been shortened

Returns: the shortened version of the text

Example:

            $text = "Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer scelerisque massa ipsum. Nullam massa tortor, tempus et lobortis a, consequat a tellus.";
            echo wordCut($text, 55, "...");

This example will produce the output: "Lorem ipsum dolor sit amet, consectetur adipiscing...".
