if_set, set_var, one_of

Description:

A kind of COALESCE function collection for PHP, whereas it does not simply check for NULL/not NULL but for real contents. if_set simply checks two strings and returns the first one set. set_var sets the value of the first parameter to the value of the first parameter which is set. one_of takes any number of arguments and returns the first which is set.

Syntax:

String if_set($str_a, $str_b)

Mixed one_of([$mix_a [, $mix_b [, É]]])

void set_var(&$mix_a, &$mix_b)

$str_a / $mix_a

First parameter to check for

$str_b / $mix_b

Second parameter to check for

Example:

The following example does in all three cases exactly the same: 

            $_SESSION["user"] = if_set(
                $_GET["new_user"], 
                if_set($_GET["old_user"], $_SESSION["user"])
            );

            set_var($_SESSION["user"], $_GET["old_user"]); 
            set_var($_SESSION["user"], $_GET["new_user"]);

            $_SESSION["user"] = one_of(
                 $_GET["new_user"], 
                 $_GET["old_user"], 
                 $_SESSION["user"]
            );

So, if $_GET["new_user"] is set, then that value will be used for $_SESSION["user"], else if $_GET["old_user"] is set, then that value will be used instead. And if none of these is set, then the session's old value remains the same.
