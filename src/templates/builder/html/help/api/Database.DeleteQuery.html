DeleteQuery

Description:

Deletes one or more tupel from a table.

Syntax:

void DeleteQuery($str_table, $str_where = "0")

$str_table

Name of the table, from which the tupel should be removed

$str_where

SQL condition to be applied for which tupel to remove.

Example:

This example - to be run within a data query of a module - removes the story with the ID passed to this function. 

            $id = func_get_arg(0);
            $this->DeleteQuery("tbl_story", "story_id='".$id."'");
