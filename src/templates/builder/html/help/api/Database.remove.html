remove

Description:

Deletes one or more tupel from a table.

Syntax:

void remove($str_table, $mix_where = "0")
void delete($str_table, $mix_where = "0")
void del($str_table, $mix_where = "0")

$str_table

Name of the table, from which the tupel should be removed

$mix_where

Either an array like: Array("fieldname1" => "fieldvalue1", "fieldname2" => "fieldvalue2", ...) or a SQL condition to be applied for which tupel(s) to remove.

The methods delete() and del() are aliases for remove().

Example:

This example - to be run within a data query of a module - removes the story with the ID passed to this function. 

            $id = func_get_arg(0);
            $this->remove("story", "story_id='".$id."'");
            // alternatively the following would do the same thing:
            // $this->del("story", Array("story_id" => $id));
