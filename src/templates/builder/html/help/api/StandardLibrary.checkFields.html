checkFields

Description:

Checks whether all named items of an array have a value (especially useful to check for valid user input after form submission)

Syntax:

bool checkFields($arr_toCheck, $arr_keys)

$arr_toCheck

array that contains the data to be checked

$arr_keys

keys in the array whose values should be checked

Returns: true, if all specified fields have a value, else false

Example:

In this example the "last_name" and the "email" fields of the array are checked whether they are populated. 

            $bol_valid = checkFields( Array(
                "first_name"=>"",
                "last_name"=>"Name",
                "email"=>"test@test.com"
             ), Array(
                "last_name", "email"
             ));

So in this case the return value of checkFields would be true. If we would include "first_name" in the second parameter, checkFields would return false, because the first_name is empty.
