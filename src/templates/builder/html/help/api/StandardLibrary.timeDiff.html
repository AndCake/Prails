timeDiff

Description:

function to display an age instead of a time

Syntax:

String timeDiff($time, $options = Array())

$time

standard unix time stamp (time in seconds from 1/1/1970)

$options

Options array (OPTIONAL)
Options include:

  • "to" => time() - date to compute the range to
  • "parts" => 1 - number of parts to display max
  • "precision" => 7 - lowest part to compute to (7 = second)
  • "distance" => TRUE - include the 'ago' or 'away' at the end (if false neither 'ago' nor 'away' appear at the end)
  • "separator" => ', ' - separates the parts
  • "period_names" => Array( 
		0=>Array("decade", "decades"), 
		1=>Array("year", "years"), 
		2=>Array("month","months"), 
		3=>Array("week", "weeks"), 
		4=>Array("day", "days"), 
		5=>Array("hour", "hours"), 
		6=>Array("minute", "minutes"), 
		7=>Array("second", "seconds")
)
    - for translational purposes
  • "less_than_1" => 'less than 1' - for translational purposes
  • "distance_names" => Array("ago", "away") - for translational purposes
  • "distance_post" => FALSE - should the distance added after or before all parts

Returns: A string expressing the time difference, e.g. "2 days, 4 hours ago".

Example:

			echo timeDiff(1234567890); // will echo something like "2 years ago"
			echo timeDiff(1234567890, Array("to" => 1234567899)); // will echo "9 seconds ago"
			echo timeDiff(1234567890, Array("to" => 1, "parts" => 4)); // will echo "3 decades, 9 years, 1 month, 2 weeks away"
