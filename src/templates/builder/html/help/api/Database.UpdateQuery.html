UpdateQuery

Description:

Update one or more tupel of a specified table and returns the number of rows updated.

Syntax:

Integer UpdateQuery($str_table, $arr_data, $str_where = "1", $bol_enclose = true)

$str_table

Name of the table to be updated

$arr_data

Data to be updated. Only entries, whose key matches one of the table's field names, are used.

$str_where

WHERE condition to decide what to update

$bol_enclose

false, if the values should not be enclosed by '.

Example:

            $id = func_get_arg(0);
            $arr_data["first_name"] = "Test";
            $arr_data["name"] = "User";
            $this->UpdateQuery("tbl_user", $arr_data, "user_id='".$id."'");
