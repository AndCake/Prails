update

Description:

Update one or more tupel of a specified table and returns the number of rows updated.

Syntax:

Integer update($str_table, $arr_data, $mix_filter)
Integer edit($str_table, $arr_data, $mix_filter)
Integer set($str_table, $arr_data, $mix_filter)

$str_table

Name of the table to be updated

$arr_data

Data to be updated. Only entries, whose key matches one of the table's field names, are used.

$str_where

Either an array like: Array("fieldname1" => "fieldvalue1", "fieldname2" => "fieldvalue2", ...) or a SQL condition to be applied for which tupel(s) to update.

The edit() and set() methods are aliases for the select() method.

Example:

            $id = func_get_arg(0);
            $arr_data["first_name"] = "Test";
            $arr_data["name"] = "User";
            $this->update("user", $arr_data, "user_id='".$id."'");
