toUTF8

Description:

this method first checks whether a string is in UTF-8 and if not converts it to UTF-8

Syntax:

String toUTF8($str_conv)

$str_conv

String to be converted

Returns: the String in UTF-8

Example:

			$str = "This might be a UTF-8 string or not...";
			
			$result = toUTF8($str);
			
			// $result will be a UTF-8 String
			echo $result;
