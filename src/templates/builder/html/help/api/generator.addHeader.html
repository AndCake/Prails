addHeader

Description:

Adds a custom string into the tag of the resulting HTML document. This applies only runlets, that have been decorated using the base template. Subsequent calls will add content to the header area, instead of replacing it.

Syntax:

void addHeader($str_header)

$str_header

The HTML content to be added to the header area of the HTML document.

Example:

The following example adds an RSS link header to the resulting HTML document: 

            $obj_gen = Generator::getInstance();
            $obj_gen->addHeader('');
