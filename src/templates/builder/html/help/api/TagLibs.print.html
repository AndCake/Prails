print

Description:

This tag is used to safely print data entered by the user. It is mainly used to avoid XSS attacks based on user input.

Syntax:

{var name}

Name of the variable to be printed in a secure manner. It's value is checked for HTML/XML sensitive characters (like < or >) and replaces them by HTML encoded versions (eg. < becomes &lt;)

Example:

This example renders a POSTed user input. 

        	<% $arr_param["test"]["message"] = $_POST["user"]["input"]; %>
