sendMail

Description:

Sends an HTML eMail with a multi-parted plain text component and optionally additional attachments.

Syntax:

bool sendMail($to, $subject, $content, $fromname, $fromaddress, $attachments = false)

$to

recipient's eMail address

$subject

subject of the eMail to be sent

$content

HTML content to be sent as eMail

$fromname

Name of sender

$fromaddress

sender's eMail address

$attachments

An Array containing a "file" and a "name" Array which hold the file path's to all files and their names. (default = false - no attachments)

Returns: true if sending of eMail was successful, else false.

Example:

		$attachments = Array(
			"file" => Array("./images/selection/mypic.jpg", "./images/selection/description/sometext.txt"),
			"name" => Array("./images/selection/mypic.jpg", "./images/selection/description/sometext.txt")
		);
		$content = "
Welcome!
This is a new mail from here.
Regards,
Me";
		sendMail("test@example.org", "You got Mail!", $content, "My Name", "myname@example.org", $attachments);

This example will send an HTML eMail to test@example.org with two attached files, one being an image and one a text file (both need to exist on the sending server).
