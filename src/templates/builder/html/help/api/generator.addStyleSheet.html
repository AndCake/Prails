addStyleSheet

Description:

addStyleSheet is used to register CSS files for the current request. Unlike the Javascript registration, the CSS files are not packed together nor minimized, they are currently only internally collected and handed over to the Generator's Template Engine agent

Syntax:

addStyleSheet($str_path, $str_media="screen", $mix_browser="all")

$str_path

the path to stylesheet file; this file does not need to be on the same server

$str_media

Target media for the stylesheet; standard CSS media formats are supported Ð like "print", "screen", "handheld", etc...

$mix_browser

defines what browser this stylesheet is for (valid values are: "all", "IE" and an Array, having a "version" key with the value of the supported IE version)

Example:

The following example will include a template, that will only be applied to users using the Internet Explorer 7: 

            $obj_gen = Generator::getInstance();
            $obj_gen->addStyleSheet(
				"templates/main/css/main.css", 
				"screen", 
				Array("version"=>7)
            );

you can specify the path to stylesheet file, what media it is for () and last but not least you can also define for what browser this stylesheet is for (supported values are "all", "IE" and Array("version"=>[IE version]) )
