createThumbnail

Description:

generates a thumbnail out of an image file

Syntax:

Boolean createThumbnail($str_src, $str_dest, $int_maxWidth = 500, $int_maxHeight = 500, $int_quality = 60)

$str_src

source image to be scaled down as thumbnail

$str_dest

destination image file name

$int_maxWidth

maximum width for the resulting thumbnail (OPTIONAL)

$int_maxHeight

maximum height for the resulting thumbnail (OPTIONAL)

$int_quality

compression quality to be used

Returns: TRUE, if generation of the thumbnail was successful, else FALSE

Example:

// will scale down a huge image to 400x300 at maximum and save it as a jpg file
createThumbnail("http://kyleabaker.com/wp-content/uploads/2008/06/wallpaper-world-map-2006-large.gif", "static/images/minimap.jpg", 400, 300);
            
// will not scale down the image further as it is already very small, but still convert it to png
createThumbnail("http://csshook.com/cssresources/wp-content/uploads/2009/03/old-world-map.jpg", "static/images/samemap.png", 520, 420);
