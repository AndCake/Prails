getInstance

Description:

The Generator class is a Singleton, and thus should only be instanced by using getInstance(). If no instance of this class yet exists, getInstance() will create a new instance and return it, else it will return the previously created instance. This method is static.

Syntax:

Generator::getInstance()

Returns: always an instance of the Generator class.

Example:

The following example will first fetch the current instance of the Generator class and then use it to set the output to be AJAX compatible. 

            $obj_gen = Generator::getInstance();
            $obj_gen->setIsAjax(true);
