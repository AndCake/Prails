SqlQuery

Description:

Sends a query to the database and returns it's result.

Syntax:

Array SqlQuery($str_query)

$str_query

The complete SQL query to be sent to the database.

Example:

            $arr_result = $this->SqlQuery("SELECT * FROM tbl_user LEFT JOIN tbl_story ON fk_user_id=user_id WHERE NOT ISNULL(photo)");

$arr_result will look like this: 

            $arr_result = Array(
                0 => Array(
                    "user_id" => "4",
                    "photo" => "test.jpg",
                    "story_id" => "",
                    "fk_user_id" => "",
	                "title" => ""
               ),
               1 => Array(
                    "user_id" => "19",
                    "photo" => "mypicture.jpg",
                    "story_id" => "25",
                    "fk_user_id" => "19",
                    "title" => "My Test Story"    
               ),
               ...
            );
