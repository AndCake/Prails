input

Description:

This tag renders a user input field.

Syntax:

{name}

name of the input to be used for submission

{type}

type of input; can be: text, password, file, checkbox, radio, select, date (default = text)

{value}

single value (for text, password, date), selected value (for select, radio), selected values (for checkbox and select box multiple; values split by ";")

{var name}

all values (for radio, checkbox, select) : Array(value : label)

{css classes to add}

classes to add

{label}

input field's label

{expression}

custom expression for required inputs (for text, password and date)

{overlabel}

overlabel to use (for text, password, date)

{size}

size to show (for select), also enables selecting multiple entries at once; for text inputs it will enable entering multiple lines of text

Example:

This example renders a select box and a textarea: 

		<@ $arr_param['values'] = Array("" => "select...", "one", "two", "three", "four"); @>
		<@ $arr_param['local']['content'] = "Hello World!\nThis is text."; @>
