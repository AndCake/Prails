setTitle

Description:

for the current runlet, set the resulting HTML title tag content (if it is fully decorated with the base template). Once it has been set, it won't be changed in subsequent calls, except it's explicitly wanted.

Syntax:

void setTitle($str_title, $bol_override = false)

$str_title

New Title for the current page

$bol_override

If set to true, it will override the page title that has been set previously.
