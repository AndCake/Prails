getUserLanguage

Description:

Retrieves the language settings preferred by the user.

Syntax:

String getUserLanguage($arr_allowedLangs, $str_defaultLang)

$arr_allowedLangs

List of RFC4646 compliant language identifiers (like "en-us" or "de-de")

$str_defaultLang

Default language that should be used, if no language preferences can be identified - also RFC4646 compliant

Returns: RFC4646 compliant language identifier

Example:

            if ( getUserLanguage(
                    Array("de-de", "es-br", "en-us", "en-uk", "en-au"), 
                    "en-us"
                 ) == "es-br" ) {
                echo "You prefer speaking spanish? Como esta usted?"
            }
