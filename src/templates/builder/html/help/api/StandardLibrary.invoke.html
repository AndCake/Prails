invoke

Description:

Invokes an event and returns it's result

Syntax:

Mixed invoke($str_event, $arr_param = null)

$str_event

name of the event to be invoked (in colon notation)

$arr_param

additional parameters that should be transmitted (OPTIONAL)

Returns: If successful it will return the event's result. If an error occured (like event not found) it will return false

Example:

            $mail = invoke("user:prepareMail", Array("user" => $_GET['user_id']));
            sendMail("test@example.org", "some subject", $mail, "My Name", "myname@example.org");

This example will send an eMail to test@example.org whereas the email's content is generated by the user:prepareMail event.
