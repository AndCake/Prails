jumpTo

Description:

Redirect the user to another page.

Syntax:

void jumpTo($str_url)

$str_url

the URL to redirect the user to (can be relative or absolute)

Returns: nothing

Example:

        	if ($_GET["test"] == "1") {
	        	jumpTo("http://www.google.com");
        	} else {
        		jumpTo("user/home");
        	}
        	die();

This example will redirect the user to Google if a GET parameter "test" is set to one, else will redirect the user to the user:home event.
