setIsAjax

Description:

Sets the current page as being an AJAX page and thus won't decorate it with the base template.

Syntax:

void setIsAjax($bol_ajax = true)

$bol_ajax

Set to true, if the page should be treated like an AJAX page and not be decorated, else false

Example:

The following code fragment shows the use of that function within an event's printer: 

			$obj_gen = Generator::getInstance();
			if ($arr_param["ajaxRequest"]) {
				$obj_gen->setIsAjax();
			}
			return $obj_gen->includeTemplate("templates/user/html/view.html", $arr_param);
