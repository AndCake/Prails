isEmbed

Description:

Checks whether a string contains HTML code.

Syntax:

bool isEmbed($str_embed)

$str_embed

Embed content to check

Returns: true , if it contains HTML code, else false

Example:

            if( isEmbed("This is just a normal, simple test embed here... ".
                    "<embed height='150' width='300' ".                             
                    "src='http://www.example.org/test.swf' wmode='window' ".                 
                    "type='application/x-shockwave-flash' />") ) {
                echo "Embedding of dynamic content is not allowed here...";
            }

The isEmbed call will evaluate to true and thus the warning text will be printed.
