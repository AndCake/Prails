hyperlink

Description:

Replaces in-text URLs by real hyperlinks.

Syntax:

String hyperlink($str_text)

$str_text

text to scan for URLs

Returns: text with all URLs replaced by real links.

Example:

            $text = "Go here: http://www.google.com/#q=The+question+of+life+the+universe+and+everything
            to see more about it...";
            echo hyperlink($text);

This example will produce: 

            "Go here: <a href='http://www.google.com/#q=The+question+of+life+the+universe+and+everything'>www.google.com</a> to see more about it..."
