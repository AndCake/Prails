doPost

Description:

Does a server-side post request to another server/site. HTTPS is supported.

Syntax:

Mixed doPost($str_url, $arr_post = Array, $user = false, $passwd = false)

$str_url

URL to POST to

$arr_post

data to be posted

$user

For HTTP authentication: the username to use to connect to the server (OPTIONAL)

$pass

For HTTP authentication: the password to use to connect to the server (OPTIONAL)

Returns: content of the server's reply

Example:

			$reply = doPost("http://www.example.org/cgi-bin/process.cgi", Array(
				"first_name"=>"Test",
				"last_name"=>"User",
				"age"=>"27",
			));

This example will post three parameters to an external server and store the server's reply in $reply.
