setDescription

Description:

for the current runlet, set the resulting description meta tag content (if it is fully decorated with the base template). Once it has been set, it won't be changed in subsequent calls, except it's explicitly wanted.

Syntax:

void setDescription($str_desc, $bol_override = false)

$str_desc

New description for the current page

$bol_override

If set to true, it will override the page's meta tag description that has been set previously.
