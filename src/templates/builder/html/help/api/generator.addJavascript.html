addJavascript

Description:

addJavaScript is used to register Javascript files for the current request. Javascript files registered in the Generator are minimized and packed together into one, static file, stored in the Javascript Cache. For every used combination of Javascript files registered, a different Javascript Cache file is created. Once this cache entry exists, no more repacking and re-minimizing is needed for this combination of Javascript files in all subsequent requests. If a Javascript file changed since the cache file has been generated, the old cache entry is removed and will be newly generated.

Syntax:

addJavaScript($str_file)

$str_file

path to the Javascript file to be registered. This file does not need to be on the same server

Example:

The following example will register the prototype.js for the current request, which can then be used within your HTML/Javascript code. 

            $obj_gen = Generator::getInstance();
            $obj_gen->addJavaScript("http://prototypejs.org/assets/2009/8/31/prototype.js");
