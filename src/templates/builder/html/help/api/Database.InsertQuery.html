InsertQuery

Description:

Inserts a tupel into the specified table and returns the ID of the new entry.

Syntax:

Integer InsertQuery($str_table, $arr_data, $bol_enclose = true, $bol_escape = true)

$str_table

The table's name the tupel should be inserted into.

$arr_data

Data to be inserted. Only entries, whose key matches one of the table's field names, are used.

$bol_enclose

false, if the values should not be enclosed by '.

$bol_escape

False, if the values should not be escaped before insertion.

Example:

This example inserts a user into the tbl_user. 

            $user_id = $this->InsertQuery("tbl_user", Array(
                "first_name" => "Test",
                "last_name" => "User",
                "email" => "tester@example.org"
            ));
