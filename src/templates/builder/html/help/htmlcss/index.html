HTML and CSS

As we have seen in the web site basics, every web site consists of some HTML and some CSS. As we have learnt previously, HTML is there to contain and structure your content, whereas CSS is used for the page's design. We're now going to take a closer look at both technologies.

The two examples H1 and P we have seen before are just a few of the many HTML tags. In fact, there are many different kinds of tags and each of them does something specific. Some of them even can only be used in conjunction with certain other tags.

What they all have in common is, that every one of them defines a certain semantic meaning to the contents it encloses and it comes with a certain default styles that we can change using CSS.
