HTML and CSS

In order to define our style for a button, we define it for all HTML tags that have the class attribute set to btn by writing before the first line: .btn {}. The . references all elements that have the class directly following the dot (in this instance "btn").

Between the curly brackets, add the line @button(rgb(165, 209, 51));. This is again a Mixin. By means of the round brackets we can provide it with options. In this case, we tell it to generate a greenish button (the color we provided it with). When we try it now, the button looks nice, but it's text is way too large.

So to change the font size, we add the third parameter to the mixin options: @button(rgb(165, 209, 51),, 16px);. All possible options and the mixin's definition can be found in the global module's styles. Because all links have are underlined by default, we tell it not to be by adding text-decoration: none;.
