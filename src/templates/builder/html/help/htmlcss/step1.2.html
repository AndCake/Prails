HTML and CSS

Now, as you can see, the text inside the P tag is still not optimally positioned on the page. We actually want it to be a part of the logo. Therefore we add a new style definition to get it into place.

The new style definition should be for that specific P tag that follows directly after the H1 tag in the HTML code. For this, there is a special notation: h1 + p {}. This is called a selector rule. Add this rule after the H1 style definition.

The attribute we define for this selector rule is: margin: -15px 0 0 74px;. This tells the P element to move 15 pixels up and 74 pixels to the left. The two 0 there are the margins for the bottom and right side, where we don't want to move.
