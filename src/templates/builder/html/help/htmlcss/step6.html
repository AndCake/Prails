HTML and CSS

There are many more things to learn about HTML. We will cover some of these things later as we move to the dynamic pages. If you want to learn more about HTML and what other tags there are, please take a look over at the Mozilla Developer Network. They have a really good reference of all the tags and their possible attributes.

For styling Prails uses an extended version of CSS, called LESS, which we used already for all our styling. There are many more things to learn here as well. For more information and details, please look at the official LESS documentation.

When you are creating your own pages, it depends on the availability of a good design. If you have one before building the page, you can often just compute such information. If not, you can use Firebug for Firefox or the Developer Console in Safari/Chrome in order to fiddle with the CSS attributes before implementing it.
