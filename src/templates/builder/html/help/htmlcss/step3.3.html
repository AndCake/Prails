HTML and CSS

When you try it out now, you'll notice, that the link's color is changing to gray if you move your mouse over it. That state is called hover. In CSS we can modify the style of that state as well. The following style definition inside the A tag will take care of it: &:hover { color: rgb(187, 135, 47); }.

But what does it do? The & at the beginning means that whatever comes next is actually not for something inside the surrounding tag (in this case something inside the A's style definition), but actually an addition to the style definition itself. The :hover is called a pseudo attribute, as it refers to a specific state (the "hover" state) of a style definition. So in fact, we define the style for the "hover" state of the A tag.
