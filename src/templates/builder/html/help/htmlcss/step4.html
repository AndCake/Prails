What is missing now, is the actual content of the page and a footer. Let's add those too. First, we declare everything we did before as being part of the header. Therefore, first go back to the Bakery home event handler and write <header> before the <div id="logo"> and move the closing HEADER tag to the very end, after the NAV tag.

In general within the output code, there is also a shorter way of wrapping something into a tag (or a set of tags for that matter): mark the text you want to enclose and press the CMD/CTRL+Shift+A key combination. Prails will ask you with what you want to enclose it with, where you - for this example - simply enter header or the respective tag you want to wrap the selection with and press OK.
