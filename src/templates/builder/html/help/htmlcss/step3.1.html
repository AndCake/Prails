Let's add those styles for the menu. First of all, go back to the module's styles. Then we'll add an entry for the logo to stay left of whatever comes next (in our case it's the NAV tag). We do this by writing: #logo { float: left; }. With #logo, we reference the tag with the ID "logo". So instead of the tag's name, here we are more specific and reference only the tag whose ID is "logo".

Okay, now we need to tell the navigation to be on the right-hand side. Therefore, we add a style definition for the NAV tag: nav { float: right; margin-top: 48px; }. This is already pretty nice, but still doesn't look the way we intended it to. To make the menu appear in one line, without the bullet points and so on, we need to change the default style of the UL within the NAV tag. So, inside the nav style definition, add a style definition for the UL tag and set the attributes list-style-type: none;, which removes the bullet points and margin: 0px;, which removes the margin it has by default.
