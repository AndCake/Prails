HTML and CSS

To complete the menu structure, we now need to tell HTML what content shall be clickable. For this, there's the <a> HTML tag, which stands for anchor.

For the A tag to work as a link, it requires an attribute, called href="[link destination URL]". Clicking the A tag's content will bring the user to the URL specified in the HREF attribute. Attributes are always written as part of the opening tag and follow the tag's name. For the A tag, this looks like that: <a href="http://abc.io/">click me</a>.

Now surround the About text with an A tag, so that it looks like this: <a href="static/about.html">About</a>.

Do the same with the other LIs, so that every LI has an A tag with the text inside. For Products & Prices, set the HREF attribute value to Bakery/products. For Store Locator set HREF to static/stores.html and for Contact Us set it to Bakery/contact. Test it.
