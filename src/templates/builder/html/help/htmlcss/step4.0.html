HTML and CSS

The actual content will come into a <section> tag just below everything else. Add that new tag. Inside it, write <div id="intro">{bakery.intro}<a href="Bakery/products" class="btn">Browse Our Cakes</a></div><div id="featured">{bakery.featured}</div><div id="news">{bakery.news}</div>. The texts inside the curly brackets (e.g. {bakery.intro} or {bakery.news}) indicate that we do not want to specify the actual content programmatically, but let a content manager choose what he/she wants to put there. This makes it easier to manually change the contents without any programming skills. These things are called Content Assets.

In the A tag for "Browse Our Cakes", you might have noticed another attribute, called "class", which we've not seen before. This defines that A tag to be part of a class of elements, that share the same thing: the class btn. We can use that in CSS to style all elements with that class. The difference between the ID attribute and the CLASS attribute is, that there can only be one tag with a certain ID, but there can many tags be in the same class.
