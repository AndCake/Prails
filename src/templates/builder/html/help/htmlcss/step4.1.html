HTML and CSS

The last thing, that is missing, is the footer of our page. Go to the end of your output code and type <footer>&copy; 2013 My Bakery Company. All Rights Reserved.</footer>. There is this strange text in there: &copy;, of which we already have seen a similar thing in our menu: &amp;. These are called HTML entities, which are used to produce certain characters, in this instance a copyright symbol. There are many other such entities. You can find a list of HTML entities over at the W3 Schools.
