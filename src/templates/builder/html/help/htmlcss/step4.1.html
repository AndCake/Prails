HTML and CSS

The last thing, that is missing, is the footer of our page. <footer>&copy; 2013 My Bakery Company. All Rights Reserved.</footer>. There is this strange text in there: &copy;, of which we already have seen a similar thing in our menu: &amp;. These are called HTML entities, which are used to produce certain characters, in this instance a copyright symbol. There are many other such entities. You can find a list of HTML entities over at the W3 Schools.

Now let's get to some styling for the new things we added in the previous step. Go back to the module styles. First of all, we want that all our main page elements, the header, the content section and the footer always are below one another. Therefore, we add to our styles at the very bottom: header, section, footer { .clearfix; }.

You might be wondering over the .clearfix, as it doesn't look like our normal [attribute]: [value] definition. In fact, this is a kind of template, called Mixin. It is used to generate a set of CSS attribute definitions. This happens automatically when it is sent to the client.
