Now that we have our logo, let's add a navigation menu for the user to navigate between the different pages that we will have in our application.

In HTML, there is a special tag for marking up the main navigation menu. It is called <nav>. Go back to the home event handler and add that NAV tag below the P tag - keep it empty for now. A NAV tag is of course not enough to give us the menu. We need to define the menu's content.

Our main menu is actually a list of links to other pages. So for lists specifically, there are other tags: <ul> and <ol>, whereby UL is used for unordered lists and OL is used for ordered lists. Both tags are dependent a sub-tag: <li>, which stands for list item. UL and OL can't be used without at least one LI.

So accordingly, add a UL tag into the NAV tag. Then insert the following LIs into the UL tag: <li>About</li>, <li>Products &amp; Prices</li>, <li>Store Locator</li> and <li>Contact Us</li>. These will become our links.
