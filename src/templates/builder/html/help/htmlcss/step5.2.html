For creating a decorator go back to the list of event handlers at the Bakery module. Then click the Add button and enter a name for the decorator into the event handler name's field: decorate. Then move all the output code from the home event handler over to the new event handler. But keep everything inside the SECTION tag in the home handler's output code.

In the decorate event handler, now remove everything inside the SECTION tag and instead write there: <c:body/>. This by itself declares the event handler as a decorator. Save it.
